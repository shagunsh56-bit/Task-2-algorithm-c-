
# internSpark Task 2: Algorithms & Problem Solving

## What it does
Implements 3 core algorithms in C++: Binary Search, Quicksort, and 0/1 Knapsack with dynamic programming.

## How to compile
g++ binary_search.cpp -o bs -std=c++17
g++ quicksort.cpp -o qs -std=c++17  
g++ knapsack.cpp -o ks -std=c++17

## Sample run
Run `./bs`, `./qs`, `./ks` to see test cases. All outputs documented in `report.md`.

## Complexity
See `report.md` for time/space complexity and test cases.
